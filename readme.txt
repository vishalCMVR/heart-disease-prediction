# ❤️ Heart Disease Prediction System

An AI-powered Heart Disease Prediction Web Application built using Machine Learning and Streamlit.

This project predicts the probability of heart disease based on patient health parameters using Logistic Regression.

---

## 🚀 Live Demo

👉 [Open Web App]https://heart-disease-prediction-dv7r77fzkdvkp23bhuyd3y.streamlit.app/

---

## 📌 Features

- Modern Streamlit UI
- Logistic Regression Model
- Probability-based Prediction
- Threshold Optimization
- Feature Scaling using StandardScaler
- Hyperparameter Tuning using GridSearchCV
- Medical Risk Analysis
- Real-time Predictions

---

## 🧠 Machine Learning Concepts Used

- Logistic Regression
- Feature Selection
- Hyperparameter Tuning
- ROC Curve
- Threshold Optimization
- Precision / Recall Tradeoff
- Confusion Matrix
- Pipeline
- Model Serialization (Pickle)

---

## 📊 Input Features

The model uses the following medical attributes:

- Age
- Gender
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol Level
- Fasting Blood Sugar
- Rest ECG Results
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression (Oldpeak)
- Slope of ST Segment
- Number of Major Vessels
- Thalassemia
